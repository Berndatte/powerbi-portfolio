# HR Analytics Dashboard
## This dashboard is used to analyze workforce composition, employee attrition and key factors influencing employee turnover.

## Dataset
IBM HR Analytics Employee Attrition Dataset.

## Key Metrics
- Total Employees
- Attrition Rate (%)
- Attrition Rate by Department
- Average Salary
- Attrition Count
- Attrition by Tenure

## Key Insights
- Sales department has highest attrition rate
- Employees working overtime show higher attrition
- Younger employees leave more frequently
- Laboratory technicians have the highest attrition rate.

## Tools used
- Power BI
- Power Query
- DAX
